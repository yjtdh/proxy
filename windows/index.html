# 教程

1.下载Clash For Windows.(7z为免安装版本,考虑到版本迭代过快,所以取消原本的安装版)点击下载

2.解压Clash For Windows,在目录下打开Clash for Windows.exe

3.在桌面右下角任务栏里点击小猫咪,打开Clash主界面.

4.点击左侧Profiles,在右侧Download from a URL处填写配置文件订阅地址并回车,软件会自动下载配置文件.

5.点击左侧Proxies查看是否成功读取节点信息.

6.点击左侧General回到软件主界面,点击System Proxy,等待十五秒即可开始访问YouTube.

桌面端和移动端URL略有不同.输入的时候别整错了.

桌面端的节点只有台湾和香港,节点少,速度可能会比较慢,建议晚高峰使用其他客户端观看.
